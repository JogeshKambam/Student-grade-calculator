Student Grade Calculator
The student grade calculator is a python program that takes a student marks and calculates the grade based on the predefined rules. It also displays an encouraging message for the student.

Objectives
*Take student name and marks as input
*Validate marks(0-100 only)
*Calculate grade using if-elif-else
*Displays motivating message
*use functions and loops properly
*Build clean,readable code

Setup & installation:
1. Install Python (version 3x)
2. Create a project folder
3. Add the following files in it:
    *arenagradecal.py
    *README.md
    *test_cases.txt
    *Screenshots
4. Run the program using:
     arenagradecal.py      

Code-Structure Explanation:
*get_marks() 
   Uses a while loop to check marks are between 0 and 100.
*calculate_grade()
  Uses if-elif-else to  decide grade and message.
*main()
  Combines input,logic and output.
